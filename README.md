### Long-Form Scientific Question Answering in Systems Biology

This repository contains the **dataset**, **code**, and **evaluation resources** for our work.

The dataset can be found at: ``src/data/thinksysbio``


## Overview

ThinkSysBio is an expert-curated dataset designed for studying **long-form scientific question answering (LFQA) in systems biology**.  
The task requires:

- multi-source evidence integration
- multi-step reasoning
- long-form scholarly report generation

The repository also includes code for running agent configurations such as **ReAct** and **ReAct-LF**, along with evaluation scripts.
